How to run main_script.py
This project provides a FastAPI-based application for uploading and viewing images and documents. It supports uploading files via API endpoints and displaying them in a simple gallery accessible through a browser.

Installation: 
- Make sure to install dependencies: pip install fastapi uvicorn

Run the application: 
 1. Start the server. Run the following command in terminal: python -m uvicorn main_script:app --reload (previously run: pip install python-multipart)

2. run: python main_script.py

3. View the Gallery Open http://127.0.0.1:8000/gallery to see uploaded images and documents.
